# 云护童心 - 儿童心理健康成长调查系统

## 项目介绍

"云护童心"是一个专为乡村儿童设计的心理健康成长调查系统，旨在通过科学的测评方法，了解儿童的心理健康状况，并提供针对性的资源和建议。

### 主要功能

- **学生测评**：儿童心理健康小调查，包含基本信息、选择题和开放性问题
- **老师管理**：班级数据统计、特殊儿童管理、红色任务布置、用户记录管理
- **管理员管理**：所有数据管理、资源管理、密码修改
- **AI分析**：基于问卷结果的智能分析和建议
- **资源包**：根据测评结果推荐的心理健康资源

## 技术栈

- **前端**：HTML5, CSS3, JavaScript
- **数据存储**：IndexedDB (浏览器本地存储)
- **AI分析**：阿里云通义千问API
- **响应式设计**：适配PC端和移动端

## 使用说明

### 学生测评
1. 选择"学生"角色
2. 点击"开始使用"
3. 填写测评问卷
4. 提交后查看结果和推荐资源

### 老师管理
1. 选择"老师"角色
2. 点击"开始使用"
3. 填写老师测评问卷
4. 进入老师管理界面，查看班级数据和管理特殊儿童
